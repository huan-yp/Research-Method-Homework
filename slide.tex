\begin{frame}
    \frametitle{CLIP 的训练——对比学习}
    \begin{columns}
        \column{0.5\textwidth}
        \begin{litemize}
            \item 训练方法：图文对比 (ITC)
            \item 优化函数：对比损失
            \item 为什么有效？
            \begin{litemize-p}
                \item 损失函数就是考试成绩
                \item 倒逼模型学习一个合理的图文表示
                \item 不相关的推远，相关的拉近
                \item 大规模数据+大模型
            \end{litemize-p}
        \end{litemize}
        \column{0.5\textwidth}
        \begin{figure}[htpb]
            \begin{center}
                \includegraphics<1->[width=1\linewidth]{pic/CLIP-ITC.png}

\end{frame}

    \begin{columns}

            \item 作为其他任务的编码器。\\

            \item 为什么能迁移？
